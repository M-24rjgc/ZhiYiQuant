# ZhiYiQuant Desktop 软著申报附件包说明

本文件用于将当前项目材料整理成“可提交附件包”的形式。建议你最终提交时，将下列文件分别导出为 PDF 或打印件，并统一放入一个申报文件夹。

## 一、建议附件结构

1. 封面
   使用 `00_cover_template_cn.md`
2. 软件基本信息说明
   使用 `overview_cn.md`
3. 功能说明书
   使用 `feature_spec_cn.md`
4. 技术架构说明
   使用 `technical_spec_cn.md`
5. 操作手册
   使用 `operation_manual_cn.md`
6. 申请检查清单
   使用 `application_checklist_cn.md`
7. 源程序鉴别材料
   使用 `source_excerpt_v2.2.0.txt`

## 二、建议导出名称

- `附件1-软件基本信息说明.pdf`
- `附件2-功能说明书.pdf`
- `附件3-技术架构说明.pdf`
- `附件4-操作手册.pdf`
- `附件5-申请检查清单.pdf`
- `附件6-源程序鉴别材料.pdf`
