# ZhiYiQuant Desktop 功能说明书

## 1. 系统首页与看板

软件提供统一桌面入口，用户登录后可以查看仪表盘、市场概览、策略状态和各功能模块入口。系统首页用于承接后续的指标分析、AI 分析、交易助手和持仓监控等功能。

## 2. 指标分析功能

- 支持市场品种的 K 线与技术指标展示
- 支持用户编写和维护 Python 指标代码
- 支持参数化指标定义与复用
- 支持图表上展示买卖信号和回测结果

## 3. AI 分析功能

- 支持针对指定市场和标的发起快速 AI 分析
- 支持输出方向判断、风险提示、止盈止损建议等结果
- 支持分析结果历史记录、分页查看与删除
- 支持根据界面语言输出中英文分析结果

## 4. 交易助手功能

- 支持创建、编辑、启动、停止和删除策略
- 支持单标的策略和横截面策略配置
- 支持回测与实盘相关参数设置
- 支持多种交易模式及信号流转

## 5. 持仓与投资组合功能

- 支持用户维护手工持仓记录
- 支持实时价格获取与盈亏计算
- 支持分组管理、备注、标签和历史平仓记录
- 支持资产监测与后续自动化监控能力

## 6. 市场数据功能

- 支持加密货币、股票、外汇等多市场数据能力
- 支持实时价格、K 线、基础面数据和全局市场概览
- 支持代理配置与本地环境适配

## 7. 用户与系统管理功能

- 支持用户登录、权限管理、个人资料管理
- 支持管理员查看用户信息和系统概览
- 支持系统设置、接口配置和通知设置

## 8. 桌面化能力

- 支持 Tauri 2.x 桌面应用封装
- 支持前端界面与本地 Python sidecar 协同运行
- 支持通过桌面应用动态分配后端端口
- 支持 Windows 安装包构建与分发
