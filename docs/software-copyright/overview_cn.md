# ZhiYiQuant Desktop 软件基本信息说明

## 1. 软件名称

- 中文名称：智弈量化桌面版软件
- 英文名称：ZhiYiQuant Desktop
- 版本号：V2.2.0

## 2. 软件定位

ZhiYiQuant Desktop 是一款面向量化交易研究、策略开发、回测分析、交易执行与持仓监控的一体化桌面应用软件。软件采用本地优先设计，支持在用户自己的设备上完成市场数据采集、指标编写、策略回测、AI 分析、交易配置及结果查看，降低对第三方 SaaS 平台的依赖。

## 3. 目标用户

- 量化交易研究员
- 独立交易员
- 金融数据分析工程师
- 需要本地化部署的投资研究团队

## 4. 开发目的

本软件旨在提供一个本地可控、功能完整、支持多市场和智能分析能力的量化交易平台，使用户能够在同一套软件中完成从行情查看、策略设计、回测验证到交易执行和资产监测的完整流程。

## 5. 运行环境

- 桌面应用框架：Tauri 2.x
- 前端界面：Vue 2、Ant Design Vue
- 后端服务：Python Flask
- 本地数据存储：SQLite
- 支持系统：Windows 优先，兼容 macOS 和 Linux 打包链路

## 6. 软件特色

- 桌面本地化交付，支持独立安装和运行
- 可视化 Python 指标与策略分析
- 多市场行情接入与监测
- AI 快速分析与历史记忆能力
- 策略回测、交易执行与持仓跟踪一体化
