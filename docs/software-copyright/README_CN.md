## 使用建议

- 先核对本目录四份说明文档中的软件名称、版本号、著作权人名称是否与你实际申报主体一致。
- 如需提交源码鉴别材料，优先使用本目录内现成的 `source_excerpt_v2.2.0.txt`。
- 若后续代码继续更新，可重新运行生成脚本，保持材料与代码版本同步。
