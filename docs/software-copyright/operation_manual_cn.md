# ZhiYiQuant Desktop 操作手册

## 1. 安装与启动

### Windows 安装

1. 打开安装包 `../../src-tauri/target/release/bundle/msi/ZhiYiQuant_2.2.0_x64_en-US.msi`
2. 按安装向导完成软件安装
3. 双击桌面快捷方式或开始菜单中的 `ZhiYiQuant`

### 开发模式启动

1. 启动前端开发服务：在 `../../quantdinger_vue` 中执行 `npm run serve`
2. 在 `../../src-tauri` 中执行 `cargo tauri dev`
3. 程序会自动启动本地 Python sidecar，并连接桌面前端

## 2. 登录与进入系统

1. 启动应用后进入登录页
2. 输入系统账号与密码
3. 登录成功后进入首页，看板、分析、交易和持仓模块会显示在左侧菜单中

## 3. 指标分析

1. 打开“指标分析”页面
2. 选择市场和标的
3. 查看 K 线、指标曲线与信号标记
4. 编辑或保存指标代码
5. 根据需要执行回测并查看结果

## 4. AI 分析

1. 打开“AI 分析”页面
2. 从自选标的中选择分析对象
3. 点击“开始分析”
4. 查看 AI 输出的方向判断、理由、风险和建议价位
5. 如有需要，可进入历史记录查看以往分析

## 5. 交易助手

1. 打开“交易助手”页面
2. 新建策略并配置市场、标的、周期和参数
3. 保存策略后可启动或停止运行
4. 查看策略状态、执行记录与相关数据

## 6. 持仓监控

1. 打开“资产监控/投资组合”页面
2. 录入手工持仓信息
3. 系统自动抓取价格并计算盈亏
4. 可按分组查看、编辑或平仓归档

## 7. 数据文件说明

桌面模式下，数据库默认保存在系统用户目录下的 `ZhiYiQuant` 数据目录中。这样在安装后即可独立运行，不依赖外部数据库服务。

## 8. 常见问题

- 若桌面启动后无法登录，先确认本地 sidecar 是否被安全软件拦截
- 若行情接口无法访问，可检查代理配置和网络环境
- 若需要重新生成安装包，可按技术说明中的构建链路重新执行
